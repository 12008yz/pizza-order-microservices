# Конфиг карточек карусели (Frame3)

Файл `cards-config.json` задаёт размеры и отступы карточек тарифов в карусели на странице провайдеров.

**Без Docker:** редактируйте `frontend/public/cards-config.json` и обновите страницу в браузере.

**С Docker:** редактируйте `config/cards-config.json` на хосте — контейнер монтирует этот файл в `public/cards-config.json`. Перезапуск Docker не нужен, достаточно обновить страницу в браузере.

## Поля

| Поле              | Описание                                              | По умолчанию |
| ----------------- | ----------------------------------------------------- | ------------ |
| `cardWidth`       | Ширина одной карточки (px)                            | 360          |
| `cardMinHeight`   | Минимальная высота карточки (px)                      | 560          |
| `gap`             | Зазор между карточками (px)                           | 5            |
| `paddingLeft`     | Отступ слева внутри карусели (px)                     | 20           |
| `paddingRight`    | Отступ справа внутри карусели (px)                    | 20           |
| `peekRight`       | Сколько пикселей следующей карточки видно справа (px) | 7            |
| `containerMargin` | Отступ карусели от краёв экрана слева и справа (px)   | 4            |

Ширина видимой области карусели вычисляется как: `paddingLeft + cardWidth + gap + peekRight`.
